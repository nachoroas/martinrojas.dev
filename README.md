# Personal Portfolio & Engineering Blog

This repository contains my personal portfolio and engineering blog.

It serves as a public showcase of:
- Professional, academic, and personal projects
- Technical writing and case studies
- Teaching materials and learning resources
- My background and interests as a software engineer

The site is built with Next.js and deployed as a static site.

## 🌐 Live Site

👉 https://your-domain.dev

---

## 🧭 Sections

- **Inicio** – Short introduction and featured work
- **Blog** – Technical posts, notes, and project write-ups
- **Proyectos** – Selected professional, academic, and personal projects
- **Docencia** – Teaching materials, labs, and resources
- **Contacto** – Ways to get in touch

---

## 🧱 Tech Stack

- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: MDX (blog & teaching), JSON (projects)
- **SEO**: Metadata, OpenGraph, sitemap, robots.txt
- **Hosting**: Vercel

This site is static-first and does not rely on a custom backend or API.

---

## 📁 Content Structure

- `data/projects.json` – Projects and case studies
- `content/blog/` – Blog posts (MDX)
- `content/teaching/` – Teaching and learning resources (MDX)

All content is versioned and edited directly in the repository.
